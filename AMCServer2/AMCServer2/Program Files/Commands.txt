:start     	     - Start the server
:stop       	     - Stop the server
:bind [ID]  	     - Bind the server to a client. [ID]: Client ID
:getdrives  	     - Get all drives from the bound client
:cls       	     - Clear the terminal
:setpriv [ID] [PL]   - Set privileges for an active client. [ID]: Client ID [PL]: Privilege level
:help       	     - Provides help information
